# Testing With GO

Examples to test Terratest with and without Table Driven Tests.

Reference - [Using Terratest to test your infrastructure](https://benmatselby.dev/post/terratest/)
## Usage

To get started, run the following commands:

```sh
go mod init
go mod tidy
go test -v ./...
```
